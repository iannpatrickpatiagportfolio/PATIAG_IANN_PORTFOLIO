## Add More Artworks Later

1. Put the new image inside `images/works/`.
2. In `index.html`, copy one existing `<article class="gallery-item">...</article>` block.
3. Update the image path, title, description, alt text, and `data-category` values.
4. Categories currently supported: `traditional`, `anatomy`, `character`, `design`, and `digital`.

## Publish on GitHub Pages

1. Sign in to GitHub and create a new public repository named `yourusername.github.io`.
2. Upload the contents of the `personal-portfolio` folder to the repository root.
3. Commit the files.
4. Open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)`, then save.
7. The website will be available at `https://yourusername.github.io/`.
