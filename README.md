# Personal Portfolio Website

A custom portfolio website created from scratch with semantic HTML, original CSS, and lightweight JavaScript. No frameworks or downloaded templates are used.

## Files

- `index.html` — website structure and content
- `style.css` — complete responsive design
- `script.js` — mobile navigation, scroll effects, active links, and contact form behavior
- `images/` — original SVG profile illustration and project previews

## Customize Before Submission

Search the project files for these placeholders and replace them:

1. `your.email@example.com`
2. `yourusername`
3. `Your School or University`
4. `Bachelor's Degree / College Program`
5. Certification and achievement examples
6. Project details, if your real projects are different

For a real professional photo, replace `images/profile-illustration.svg` with your photo and update the image path in `index.html`.

## Publish on GitHub Pages

1. Sign in to GitHub and create a new public repository named `yourusername.github.io`.
2. Upload every file and the `images` folder to the repository root.
3. Commit the files.
4. Open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)`, then save.
7. Your portfolio URL will be `https://yourusername.github.io/`.

## Local Preview

Open `index.html` in a browser. For the best preview, use a local server such as the VS Code Live Server extension.
